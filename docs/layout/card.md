## 参数说明

### Card 参数

| 参数    | 说明               | 类型    | 可选值                           | 默认值   |
| ------- | ------------------ | ------- | -------------------------------- | -------- |
| type    | _卡片组件类型选择_ | string  | _`card`_,_`full`_                | _`card`_ |
| bgColor | _卡片背景色设置_   | string  | 参考文档 [默认色](/home/color)   | -        |
| shadow  | _卡片阴影设置_     | boolean | -                                | _`true`_ |
| title   | _卡片标题_         | Text    | 参考文档 [Text 组件](/base/text) | _`{}`_     |
